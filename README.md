## 🚀 Running the Documentation Site Locally

These notes are built using [VitePress](https://vitepress.dev/). To run the interactive documentation site locally:

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm docs:dev
```
